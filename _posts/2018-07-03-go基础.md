### 前言

Go 语言最主要的特性：

自动垃圾回收、更丰富的内置类型、函数多返回值、错误处理、匿名函数和闭包、类型和接口、并发编程、反射、语言交互性

安装：

### mac下安装
```
brew install go
```
### 语言结构
Go语言的基础组成有以下几个部份：
+ 包声明
+ 引入包
+ 函数
+ 变量
+ 语句 & 表达式
+ 注释

首先来个万能的Hello,world

```
package main

import "fmt"

func main() {
	/* 这是我的第一个简单的程序 */
	fmt.Println("Hello, World!")
}
```
